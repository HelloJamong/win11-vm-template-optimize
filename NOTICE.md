# NOTICE

## 1. 자체 작성 코드 고지

이 저장소(`win11-vm-template-optimize`)의 문서, PowerShell 스크립트, 배치 파일, 설정 예시는 Windows 11 VM 마스터 템플릿 생성을 위해 자체 작성한 자료입니다.

본 저장소의 자체 작성 코드에는 `LICENSE` 파일에 포함된 MIT License가 적용됩니다.

## 2. 참고 프로젝트

이 저장소는 다음 공개 프로젝트의 아이디어, 구성 방식, 최적화 방향을 참고했습니다. 원본 소스를 그대로 복사하거나 포함하지 않았으며, 이 저장소의 목적에 맞게 재구성했습니다.

### Win11Debloat

- 프로젝트: Win11Debloat
- 저장소: [프로젝트 바로가기](https://github.com/Raphire/Win11Debloat)
- 라이선스: MIT License
- 참고 범위:
  - Windows 10/11 기본 앱 제거 후보 분류
  - 원치 않는 소비자 기능 및 개인정보 관련 설정을 점검하는 방향
  - Copilot, Bing 검색, 광고성 구성요소 등 사용자 경험 조정 관점
- 포함 여부:
  - 원본 PowerShell 소스는 포함하지 않습니다.
  - 본 저장소의 스크립트는 별도 구조와 옵션 체계로 작성되었습니다.

### WinUtil

- 프로젝트: WinUtil
- 저장소: [프로젝트 바로가기](https://github.com/ChrisTitusTech/winutil)
- 라이선스: MIT License
- 참고 범위:
  - Windows 최적화 항목을 선택형 옵션으로 분리하는 구성 방식
  - 앱 제거, 서비스 조정, 정책 설정을 운영자가 선택하도록 하는 방향
  - 사용자 친화적인 설명과 경고를 제공하는 문서화 방식
- 포함 여부:
  - 원본 PowerShell 소스는 포함하지 않습니다.
  - 본 저장소의 스크립트와 문서는 Windows 11 VM 템플릿 운영 목적에 맞게 자체 작성되었습니다.

### Virtual-Desktop-Optimization-Tool

- 프로젝트: Virtual-Desktop-Optimization-Tool
- 저장소: [프로젝트 바로가기](https://github.com/The-Virtual-Desktop-Team/Virtual-Desktop-Optimization-Tool)
- 참고 범위:
  - 가상 데스크톱/VDI 환경에서 성능, 정책, 서비스, 예약 작업을 분류해 적용하는 운영 개념
  - 템플릿 이미지 최적화 전후 검증이 필요하다는 운영 관점
- 포함 여부:
  - 직접 코드, 설정 파일, 원본 스크립트는 포함하지 않습니다.
  - 본 저장소는 VDOT의 코드를 재배포하지 않으며, 개념과 구성 아이디어만 참고했습니다.
